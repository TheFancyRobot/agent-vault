# Vault Refresh

Refresh all Agent Vault home notes.

Call the `vault_refresh` tool to:
- Rebuild the bugs index from bug note frontmatter
- Rebuild the decisions index from decision note frontmatter
- Refresh Active Context with current focus, blockers, and critical bugs

Run this after creating or updating bug, decision, phase, or step notes.
